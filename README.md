## 📦 安装与启动

| 方式               | 说明                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------ |
| **安装版**（推荐） | 运行 `POE2 Map Guide Setup 1.0.0.exe` → 按向导完成安装 → 从开始菜单/桌面快捷方式启动 |
| **便携版**         | 将 `POE2 Map Guide 1.0.0.exe` 复制到任意位置 → 双击运行                              |

## 📱 浮窗模式

游戏时的精简置顶界面，实时显示当前地图任务。

### 开启与关闭

- 点击右上角 **📱 浮窗** 开关
- 或点击浮窗上的 ✕ 按钮关闭

### 浮窗控制

| 按钮    | 功能                  |
| ------- | --------------------- |
| 📌      | 切换置顶              |
| −       | 最小化浮窗            |
| □       | 折叠/展开内容         |
| A+ / A- | 增大/减小字体         |
| ← →     | 切换上一张/下一张地图 |
| ✕       | 关闭浮窗              |

> **提示**：拖动浮窗顶部标题栏可移动位置，浮窗会自动吸附到屏幕边缘。

## 🔍 自动地图检测

应用自动读取 POE2 客户端日志，进入新地图时自动切换任务列表。

### 首次配置

1. 点击右上角 **⚙️ 设置**
2. 在"游戏日志路径"区域添加 POE2 日志文件
3. 默认路径：`C:\Program Files (x86)\...\Path of Exile 2\logs\Client.txt`

### 多客户端支持

支持配置多个日志路径（国际服/国服），激活所需路径即可自动监控。

## ✏️ 编辑模式

每个章节和地图都有独立的 **开始编辑** 按钮，开启后即可修改内容。

### 章节编辑

开启后可在章节内进行：

- **添加/删除地图**
- **编辑攻略** ✎ — 修改章节攻略和 Boss 攻略
- **添加图片** 📷 — 为章节添加参考图片
- **拖拽排序** — 拖动地图调整顺序

### 地图编辑

开启后可在地图内进行：

- **添加/删除任务**
- **修改任务** ✎ — 编辑描述、优先级标签、奖励标签
- **添加图片** 📷 — 关联地图截图或笔记图片
- **编辑地图名** ✎ — 修改名称
- **编辑攻略** ✎ — 修改探索说明
- **拖拽排序** — 拖动任务调整顺序

### 使用场景

- 记录个人探索发现
- 添加自定义笔记和截图
- 调整任务优先级顺序

## 🖼️ 图片功能

### 添加图片

编辑模式下，点击 **📷 添加图片** 按钮：

1. 选择本地图片文件（支持 jpg / png / gif / webp / bmp）
2. 图片以缩略图形式显示在标题旁
3. 可重复添加多张图片

### 查看图片

点击缩略图打开图片查看器：

| 操作         | 效果         |
| ------------ | ------------ |
| 滚轮         | 缩放图片     |
| 拖拽         | 移动图片位置 |
| 点击空白区域 | 关闭查看器   |

### 删除图片

编辑模式下，点击缩略图右上角的 🗑 按钮即可删除。

## 💾 数据管理

### 导入/导出进度

| -------- | --------------------------------------- |

## ❓ 常见问题

<details>
<summary>浮窗无法显示？</summary>

1. 关闭应用后重新打开
2. 检查是否被其他窗口遮挡
3. 重启电脑后再试
</details>

<details>
<summary>地图检测不自动切换？</summary>

1. 确认设置中日志路径正确
2. 检查 POE2 是否正在运行并生成日志
3. 尝试重新选择日志文件
</details>

<details>
<summary>进度丢失怎么办？</summary>

1. 检查是否清理了浏览器缓存（应用使用 localStorage）
2. 建议定期使用"导出缓存"功能备份
</details>

<details>
<summary>便携版与安装版的区别？</summary>

便携版无需安装，但不会自动注册文件关联和开始菜单快捷方式。两者功能完全一致。

</details>

## 💡 最佳实践

### 开荒流程

```
游戏前   ─→  启动应用，开启浮窗
进入地图 ─→  浮窗自动显示该地图任务
完成任务 ─→  在浮窗中勾选
切换地图 ─→  自动跟随或手动导航
下线前   ─→  导出缓存备份进度
```

### 浮窗位置建议

推荐放置在屏幕顶部中央或角落，避免遮挡角色状态栏、技能栏、小地图。
