## 附录：拍摄/录屏清单

1. **基础素材**
   - 软件主界面完整录屏（1920×1080，深色/浅色主题各一套）
   - 浮窗模式与游戏分屏画面（可用窗口化游戏录制）
   - 地图自动检测过程（需准备 POE2 游戏日志环境）

2. **关键动作（建议放慢录制，后期加速）**
   - 勾选任务、进度条变化
   - 一键清查弹窗打开与点击跳转
   - 浮窗透明度/字体颜色调节

- 编辑模式下添加/删除/拖拽操作

3. **音效建议**
   - 主 BGM：电子/史诗风格，节奏感强
   - 功能点击：清脆 UI 反馈音
   - 标记完成：轻快的“完成”音效
   - 结尾：骤停 + 单音收束

4. **字幕规范**
   - 字体：思源黑体 / Montserrat 加粗
   - 颜色：白色为主，关键词用 `#ff9800`（橙色）高亮
   - 位置：画面下方 1/3 处，带轻微阴影
